# Task Manager

A simple and scalable Task Manager backend built with **Node.js**, **Express**, **MongoDB**, and **Mongoose**.  
This project is organized using **Gitflow** for a clean and professional development workflow.

---

## Features

- Create, read, update, and delete tasks
- MongoDB integration with Mongoose
- Task model with timestamps
- Environment-based configuration
- Clean backend structure
- Gitflow branch strategy

---

## Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **dotenv**
- **cors**
- **nodemon**
